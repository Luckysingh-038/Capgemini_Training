1. How does the object-oriented nature of Java help in software development?

Java follows the object-oriented programming (OOP) approach, which helps in organizing code in the form of classes and objects.
OOP concepts like encapsulation, inheritance, polymorphism, and abstraction make the software easier to design,
understand, and maintain. Code reusability reduces development time, and modularity helps teams work on large projects efficiently.

2. Why is Java considered a simple language compared to C and C++?

Java is considered simple because it removes complex features found in C and C++, such as pointers,
manual memory management, multiple inheritance, and operator overloading. Java provides automatic garbage collection,
a clear syntax, and built-in libraries, which makes it easier for developers to write, debug, and maintain programs.

3. How does Java provide security at run time?

Java provides runtime security using the Java Virtual Machine (JVM). The bytecode verifier checks the code
before execution, and the class loader prevents unauthorized classes from being loaded. Java also uses
Security Manager and access control mechanisms to restrict illegal operations, making Java programs safe from viruses and malicious code.

4. What makes Java a robust language?

Java is considered robust because it has strong memory management, automatic garbage collection,
and exception handling. It avoids common programming errors like memory leaks and crashes. Compile-time and runtime error
checking also help in creating reliable and stable applications.

5. How does Java achieve high performance?

Java achieves high performance through the Just-In-Time (JIT) compiler, which converts frequently
used bytecode into native machine code at runtime. This improves execution speed. Efficient memory management
and optimized JVM architecture also contribute to better performance.

6. What do you mean by portability in Java?

Portability in Java means ‚ÄúWrite Once, Run Anywhere.‚Äù Java programs are compiled into bytecode,
which can run on any system that has a JVM, regardless of the operating system or hardware. This makes Java applications platform-independent.

7. Why does Java have strong industries and job opportunities?

Java is widely used in enterprise applications, web development, mobile applications (Android),
banking systems, cloud computing, and big data. Many large companies rely on Java for building secure and scalable applications.
Due to its wide usage and long-term stability, Java offers strong job opportunities across industries.
